# Compilador de Intenciones

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad de Compilador de Intenciones para OpenClaw.

## Qué Resuelve
Traduce especificaciones en lenguaje natural a casos de prueba, documentación y esqueletos de código.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque de seguridad primero
- Soporte para reversión

## Uso
/intent-compiler

## Ejemplo
```
Analizar código para problemas de programación.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
